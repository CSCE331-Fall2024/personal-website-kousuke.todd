I used Verilog to synthesize the hardware and create test benches.

Verilog

As a computer engineer, Verilog is the tool that I use the most for creating hardware. Using behavioral Verilog, which is very similar to programming, it can create large scale hardware that you do not need to design yourself. Synthesizers will take behavioral Verilog, and create a digital circuit from it. This makes designing hardware efficient, due to not having to design the smaller building blocks of your design. Using Verilog, I have created a traffic light controller and CPU.
